Завдання – Створити простий WPF-додаток
Метою цього завдання є пошук програміста для заповнення нової вакансії. Програма
повинна містити наступний функціонал:
- Можливість додавання та видалення працівників до списку.
- Список усіх працівників та праціників, хто є робітником.
Ми хочемо зберігати таку інформацію про працівників:
- Ім’я (текст)
- Позиція (текст)
- Дата народження (дата)
Програма повинна складатися з:
1. Список усіх праціників (відсортованих за Ім’ям)
o Вік повинен бути вказаний замість дани народження
2. Форма для додавання праціника. Дата народження може бути звичайним текстовим
полем, так як ми очікуємо правильного введення.
3. Можливість для видалення працівника з листа.
4. Можливість для редагування свойства «Позиція» для існуючих записів
працівників.
5. Можливість для імітування почтаку та кінця роботи працівника.
Нотатки:
- Введенні данні не повинні перевірятися на помилки. Очікується коректний ввід.
- Данні не повинні зберігатися.
Оцінювання:
Завдання може бути виконано частково, часткові результати та принципи будуть прийняті
також. Фактори оцінювання роботи (відсортовано за пріоритетом):
- MVVM паттерни як View Models, Bindings, мінімум code behind
- Використання Collection View для фільтрування та сортування даних
- Читабельність коду, коментарі, найменування методів, OOP and SOLID патерни
- Живі дані, фільтри, сортування і т.п.
- Функції програми.
